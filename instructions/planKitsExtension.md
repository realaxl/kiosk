- Kits sollen genau wie einzelne Produkte behandelt werden

- wie kann die Datenstruktur erweitert werden?
- welche Anpassung in der Admin-Oberfläche benötigen wir?

- zusätzlich: Kits haben immer
  - eine Beschreibung

  - eine Notiz / note
